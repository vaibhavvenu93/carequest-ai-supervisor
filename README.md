# CareQuest AI Supervisor

AI supervision layer for neurodivergence care workforce training.

## Purpose

This repo explores how AI can support learner feedback, simulation review, coaching summaries, and supervisor-style guidance.

## Core Thesis

One of the biggest bottlenecks in care workforce training is supervision.

AI can help scale supervision by reviewing simulation responses, identifying competency gaps, and generating coaching feedback.

## What The Supervisor Reviews

- learner response
- scenario context
- empathy
- safety
- observation
- reinforcement
- communication
- next steps
- documentation readiness

## Outputs

- coaching summary
- competency feedback
- strengths
- growth areas
- improved response
- recommended next mission

## Long-Term Vision

AI-assisted supervision infrastructure for care workforce training.
